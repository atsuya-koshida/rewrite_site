よくある質問

お客様からよくいただく質問をまとめました

サービス全般について

Q 運動初心者でも大丈夫ですか？ +
A

全く問題ありません。大歓迎ですので、安心してお問い合わせください。

Q オンラインとハイブリッドの違いは？ +
A

オンラインはフルリモートで行い、チャットと週1回のオンラインコーチングを通して習慣を変えていきます。ハイブリッドでは、それに加え週1回の対面のパーソナルトレーニングを実施します。

Q 3ヶ月で本当に変われますか？ +
A

正しい方法で継続すれば必ず変わります。習慣形成には一般的に21〜66日かかると言われているので、あなたの習慣になるまで全力でサポートします。

Q どんな人が利用していますか？ +
A

20代から40代まで幅広い年齢層の方にご利用いただいています。会社員、経営者、主婦、など職業も様々です。共通しているのは「今度こそ本気で変わりたい」という強い想いをお持ちの方々です。

Q カウンセリングは本当に無料ですか？ +
A

はい、2時間完全無料です。強引な勧誘は一切ありませんので、安心してご利用ください。あなたの現状と目標をじっくりお聞きし、最適なプランをご提案します。その場で契約を迫ることはありません。

Rewrite習慣改変プログラムについて

Q 仕事が忙しくて時間が取れるか心配です +
A

忙しい方こそ効果的です。1日わずか10-30分から始められ、あなたのライフスタイルに合わせてプログラムを調整します。時間がない中でも継続できる仕組み作りがRewriteの得意分野です。

Q オンラインとハイブリッドはどちらがおすすめですか？ +
A

東京・埼玉エリアにお住まいで、より短期間で結果を出したい方はハイブリッドプランをおすすめします。地理的な制約がある方や、コストを抑えたい方はオンラインプランでも十分な効果が期待できます。

Q チャットサポートはどんな内容に対応していますか？ +
A

運動・食事・生活習慣に関する質問はもちろん、モチベーション低下時の相談、日々の進捗報告、困ったことがあった際の対処法など、幅広く、できる事は全て対応します。24時間以内に必ず返信いたします。

Q 栄養指導の内容を教えてください +
A

個人の目標に合わせた食事アドバイスを行います。極端な食事制限ではなく、持続可能な食習慣を身につけることを重視します。具体的なメニュー提案、外食時のアドバイス、コンビニ食の選び方なども含まれます。

パーソナルトレーニングについて

Q トレーニング初心者でも大丈夫ですか？ +
A

はい、もちろんです。基本的な動作から丁寧に指導いたします。あなたのペースに合わせて段階的にレベルアップできるようサポートしますので、安心してご参加ください。

Q どのくらいで効果が出ますか？ +
A

個人差はありますが、多くの方が2-3週間で体の変化を実感し始めます。見た目の変化は1-2ヶ月、大きな変化は3-6ヶ月程度で現れることが一般的です。

Q 怪我をしていても受けられますか？ +
A

軽度の怪我であれば、その部位を避けたメニューで対応可能です。ただし、重度の怪我や医師からの運動制限がある場合は、医師の許可を得てからの参加をお願いしています。

Q 食事指導も受けられますか？ +
A

パーソナルトレーニング単体では食事指導は含まれませんが、セッション時のアドバイスは可能です。詳細な栄養指導をご希望の場合は、Rewrite習慣改変プログラムをおすすめします。

Q どんなトレーニング内容ですか？ +
A

基本種目をできる状態にする為のパーソナルなアプローチを行い、目安として3ヶ月で1人で自信を持ってジムに通えるレベルを目指し、指導します。

料金・支払いについて

Q 支払い方法は何がありますか？ +
A

銀行振込、クレジットカードに対応しています。分割払いも可能ですので、お気軽にご相談ください。

Q モニター募集はいつまでですか？ +
A

限定5名に達し次第終了となります。お早めにお申し込みください。モニター価格はRewriteハイブリッドプランを通常価格の50%OFFでご提供しています。

Q 予約の変更・キャンセルはできますか？ +
A

24時間前までのご連絡で無料での変更・キャンセルが可能です。それ以降は1セッション分消化扱いとなります。

Q 入会金や初期費用はかかりますか？ +
A

入会金や初期費用は一切かかりません。表示されている月額料金のみでご利用いただけます。

実施・アクセスについて

Q 対面トレーニングはどこで行いますか？ +
A

レンタルジムを利用して行います。都内（渋谷・目黒・恵比寿・新宿・池袋など）、埼玉を中心に幅広く対応しております。場所は柔軟に対応させていただきますので、お気軽にお問合せください！

Q 営業時間外でも対応してもらえますか？ +
A

チャットサポートは24時間以内に返信、緊急時は可能な限り対応いたします。セッション時間は7:00-23:00の間で柔軟に調整可能です。

Q オンラインコーチングセッションに必要なものは？ +
A

安定したインターネット接続、カメラ・マイク付きデバイス（PCを推奨）があれば参加可能です。
